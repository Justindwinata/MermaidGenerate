# Checklist Video Demo MermaidGenerate

## Sebelum Recording

- [ ] Buka Google Colab.
- [ ] Buka `MermaidGenerate_Mindmap_Venn_Finetuning_WebUI.ipynb`.
- [ ] Set runtime ke GPU.
- [ ] Pastikan koneksi internet stabil.
- [ ] Siapkan dataset:
  - `datasets/expanded/mixed_mindmap_venn_expanded_500.jsonl`
  - `datasets/expanded/mixed_mindmap_venn_expanded_1000.jsonl`
  - `datasets/curated/mixed_mindmap_venn_curated.jsonl`
- [ ] Siapkan prompt Mind Map:
  - `Buat mind map tentang strategi belajar AI untuk mahasiswa informatika.`
- [ ] Siapkan prompt Venn:
  - `Buat diagram Venn tentang Instagram, TikTok, dan WhatsApp marketing.`
- [ ] Buka README atau `SUBMISSION_README.md`.
- [ ] Buka `docs/VIDEO_DEMO_SCRIPT_DETAILED.md`.
- [ ] Pastikan screen recorder aktif.
- [ ] Cek microphone dan audio.
- [ ] Tutup tab yang tidak relevan.
- [ ] Zoom browser agar teks terlihat jelas.
